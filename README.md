# marketplace-api
REST API for marketplace microservice

# Features
  - Moderator
  - E-commerce API(Buy and Sell)
  - Incentives
  - Referral
  - Sustainability
  - Premium user (Buyer and Seller)

# Technologies used
  - Spring boot
  - Spring mvc
  - Spring data jpa
  - Hibernate validator
  - Spring security
  - Mapstruct
  - Lombok
  - Sofware used
      - MySQL
      - Postman
      - IntelliJ

# Run with docker compose
```dockerfile
docker compose --env-file <env_file_containing_the_email_and_app_password> up
```
