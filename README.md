# marketplace-api
API for agriculture e-commerce

# Features
  - Moderator
  - E-commerce API(Buy and Sell)
  - Incentives
  - Referral
  - Sustainability
  - Premium user (Buyer and Seller)

# Technologies used
  - Spring boot
  - Spring mvc
  - Spring data jpa
  - Hibernate validator
  - Spring security
  - Mapstruct
  - Lombok
  - Sofware used
      - MySQL
      - Postman
      - IntelliJ
   
# Run with Docker Compose
```
docker-compose up -d --build
```
